# MojePierwszeRepo

Repozytorium edukacyjne stworzone do nauki pracy z Git i GitHub.

## Co robi ten projekt?

Projekt służy do praktykowania podstawowych operacji w Git:
- praca z gałęziami (branching)
- tworzenie Issues i Pull Requestów
- współpraca z zdalnym repozytorium

## Zawartość

| Plik | Opis |
|------|------|
| `Witaj.txt` | Plik powitalny z informacjami o projekcie |
| `NowaFunkcja.txt` | Przykładowa treść dodana w gałęzi rozwojowej |

## Jak uruchomić / skorzystać

1. **Sklonuj repozytorium:**
   ```bash
   git clone https://github.com/BasiaNiedbal/MojePierwszeRepo.git
   cd MojePierwszeRepo
   ```

2. **Przełącz się na wybraną gałąź** (opcjonalnie):
   ```bash
   git checkout rozwoj    # gałąź rozwojowa
   # lub
   git checkout main      # gałąź główna
   ```

3. **Otwórz pliki** – to prosta kolekcja plików tekstowych, możesz je przeglądać w dowolnym edytorze.
